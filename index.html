Selected Works

Crafting structural logic, exploring spatial topology, and delivering deeply engaging environmental experiences.

Soulslike Level Layout

This project focuses on the quantitative analysis of Soulslike level layouts and diorama generation based on graph topological features. Using the Dying Light engine, I designed complex non-linear paths, optimized loop rates, and tested stability factors to create deeply engaging player experiences.

View Project

Starfield Survival Mod

A custom survival mode mod developed using the Starfield Creation Kit, heavily inspired by Call of Duty Zombies mechanics. Key features include iterative wave control logic, custom weapon upgrade stations, and a carefully balanced spatial loop design for intense combat scenarios.

View Project

PICO-8 Retro Minigame

A retro-style game built from scratch using the PICO-8 virtual console. This project showcases scripting of complex game loops, custom collision detection algorithms, and handcrafted sound effects and music simulated directly within the PICO-8 editor.

About Me
